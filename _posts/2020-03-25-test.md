## ✨ 为什么选择 Beautiful Jekyll？

**Beautiful Jekyll** 是一个由社区维护的高颜值 Jekyll 主题。  
它适合想在 GitHub Pages 上快速拥有**个人博客或作品集**的开发者与创作者。

优点包括：

- 💡 安装简单，仅需一行配置
- 🧭 自动生成文章目录与分页
- 🎨 支持封面图、代码高亮、评论
- 🌍 完全兼容 GitHub Pages
